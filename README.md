# Path Tracking Feature

## ภาพรวม (Overview)
ฟีเจอร์นี้มีหน้าที่ตรวจจับเส้นนำทางสำหรับผู้พิการทางสายตา (Tactile Paving) และวิเคราะห์ทิศทางของเส้นทางเดินจากภาพที่ได้จากกล้อง  
ระบบจะทำหน้าที่เป็นตัวช่วยยืนยันแนวทางเดิน และแจ้งเตือนผู้ใช้งานเมื่อเส้นทางเริ่มมีการเปลี่ยนทิศทาง

## วัตถุประสงค์ (Objective)
เพื่อเพิ่มความมั่นใจและความปลอดภัยให้กับผู้พิการทางสายตา โดยช่วยตรวจสอบทิศทางของเส้นนำทาง และแจ้งเตือนล่วงหน้าเมื่อทางเดินมีการโค้งหรือเปลี่ยนทิศ

## ขั้นตอนการทำงานของระบบ (System Workflow)
1. รับภาพจากกล้องแบบต่อเนื่อง
2. แปลงภาพเป็นรูปแบบ HSV Color Space
3. ตรวจจับเส้นนำทางสีเหลืองด้วยเทคนิค Color Segmentation
4. หา Contour ของพื้นที่ที่เป็นเส้นนำทาง
5. คำนวณจุดศูนย์กลาง (Centroid) และแกนหลักของเส้นทาง
6. เปรียบเทียบตำแหน่งของเส้นทางกับกึ่งกลางภาพ
7. วิเคราะห์และกำหนดทิศทางการเดิน

## พฤติกรรมของระบบ (System Behavior)
- หากเส้นทาง **ตรง** → ระบบจะยืนยันว่าทางเดินยังตรง
- หากเส้นทาง **โค้งไปทางซ้าย** → ระบบจะแจ้งเตือน "Turn Left"
- หากเส้นทาง **โค้งไปทางขวา** → ระบบจะแจ้งเตือน "Turn Right"
- หากตรวจพบ **จุดที่มีการเปลี่ยนทิศทางหรือทางแยก** → ระบบจะแจ้งเตือนผู้ใช้ล่วงหน้า

## ผลลัพธ์ที่คาดหวัง (Expected Outcome)
ระบบสามารถช่วยผู้ใช้งานรักษาแนวทางเดินได้อย่างถูกต้อง และเตรียมตัวก่อนถึงจุดที่ต้องเปลี่ยนทิศทางของเส้นทางเดิน
